## Overview

This guide covers essential Linux commands for monitoring system performance, processes, memory, disk usage, and network activity.

## 1. `top`

``` bash
top
```

Purpose: Displays real-time active processes and system resource usage.

What to check for:

- Load average
- CPU Usage Breakdown
- Memory Usage
- Tasks Summary
- Top Processes (Main part)

## 2. `htop`

``` bash
htop
```

Purpose: Interactive process viewer with visual system metrics.

## 3. `vmstat`

``` bash
vmstat 2 5
```

Purpose: Reports CPU, memory, swap, I/O, and process stats.

## 4. `ps`

``` bash
ps -ef
```

Purpose: Lists all running processes.

## 5. `df`

``` bash
df -h
```

Purpose: Displays disk usage per filesystem.

## 6. `du`

``` bash
du -sh /var/log
```

Purpose: Estimates directory size.

## 7. `free`

``` bash
free -h
```

Purpose: Displays memory usage.

## 8. `uptime`

``` bash
uptime
```

Purpose: Shows system uptime and load average.

## 9. `netstat`

``` bash
netstat -tulnp
```

Purpose: Lists active ports and connections.

## 10. `lsof`

``` bash
lsof -i :22
```

Purpose: Lists open files and network connections.

## 11. `journalctl`

``` bash
journalctl -xe | less
```

Purpose: View system logs.

## Summary

These commands form the foundation of proactive Linux system
diagnostics.
